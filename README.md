#### Commands

```bash
# install ArgoCD in k8s
kubectl create namespace argocd
kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/stable/manifests/install.yaml

# access ArgoCD UI
kubectl get svc -n argocd
kubectl port-forward svc/argocd-server 8080:443 -n argocd

# login with admin user and below token (as in documentation):
1.	kubectl get secret argocd-initial-admin-secret -n argocd -o yaml (it is an encoded)
2.	then type echo “Password” | base64 –decode (it will generate the decrypted password) 
3.	if it is not working, then copy the password and paste it to https://www.base64decode.org/ for decoding.

# you can change and delete init password

```
